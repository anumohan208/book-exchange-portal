The Book Exchange Portal is a web-based application that allows users to exchange, lend, or request books within a community. The platform promotes reusability of books by enabling users to list books they own and connect with others who are interested in borrowing or exchanging them.

* Features

User registration and authentication

Add, update, and delete book listings

Browse and search available books

Request books from other users

Approve or reject book exchange requests

Track exchange status (requested, approved, completed)

User-friendly and responsive UI
